git

  1. Crie um repositorio no GitHub com memso nome seu projeto
  2. Com terminal de git init
  3. Altera O Nome da branch para main
  4. Adicione os arquivos com git add . e faça o commit com git commit -m "fist commit"
  5. Conecte ao repositorio remoto com git remote add origin main "link
  6. Envie os arquivos com git push origin main
  7. No GitHub, va em Settings
